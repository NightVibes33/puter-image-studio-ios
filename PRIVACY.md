# Privacy Policy

Image Studio sends prompts and generation settings to the configured image generation API only when you tap Generate. The API may forward those prompts to AI image providers to create the requested image.

Generated images are stored locally on the device after generation. The app asks for Photos add-only permission only when you tap Save.

The app does not embed a Puter login, does not store a Puter token in the iOS bundle, and does not include a public community feed in this build.

For support, open an issue at https://github.com/NightVibes33/puter-image-studio-ios/issues.
